Работа с новым css-свойством WIDTH

Этот элемент растянут на всю ширину экрана, его ширина не фиксирована. То есть не указано 5см, 100 пикселей или других единиц измерения. А указано, чтобы элемент занимал 100% ширины РОДИТЕЛЬСКОГО.

Родительским считается тот тег, внутри которого располагается другой тег, для которого мы устанавливаем стили. Для этого тега SPAN родительским является тег Р. Подробнее поясню на уроке.
